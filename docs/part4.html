当前我们使用sscanf来读取输入，sscanf有一些缺点。如果它正在读取的字符串大于它正在读取的缓冲区，则会导致缓冲区溢出并开始写入意外位置。 我们需要在每个字符串复制到 Row 结构之前检查每个字符串的长度，防止输入过长的字符串。因此，我们需要将输入中的空格过滤掉。 下面将使用 strtok（） 来做到这一点。

+  char* email = strtok(NULL, " ");

+  if (id_string == NULL || username == NULL || email == NULL) {
+    return PREPARE_SYNTAX_ERROR;

+  return PREPARE_SUCCESS;
+}
